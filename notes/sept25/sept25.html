1. Review

Today's goal was to give ourselves the tools to define recursive functions in the untyped Î»-calculus, using the factorial function on natural numbers as an illustrative example. This function can be defined as follows in Haskell: 

factorial :: Nat -> Nat
factorial n = if n == Zero then Succ Zero else mult n (factorial (pred n))
  where mult :: Nat -> Nat -> Nat
        mult Zero n = Zero
        mult (Succ m) n = plus n (mult m n)

        plus :: Nat -> Nat -> Nat
        plus Zero n = n
        plus (Succ m) n = Succ (plus n m)

        pred :: Nat -> Nat
        pred Zero = Zero
        pred (Succ n) = n
